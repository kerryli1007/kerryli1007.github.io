你负责的app日活降了5%，你会如何分析？

发表于 2019-05-07 更新于 2023-03-24 分类于 思考

本问题出自亚雄老师「互联网er读书会」每日一问。

ASK

你负责的app日活降了5%，你会如何分析？

思考，以下Enjoy

本次思考，分为4个部分：

  1. 确认APP类型
  2. 确认日活定义
  3. 确认日活下降5%的定义
  4. 分析影响日活的因素

1. 确认APP类型

首先，在分析这个问题的时候，我们需要明白，我们的APP是什么类型的APP：

  1. 工具
  2. 社交
  3. 电商
  4. 内容

为什么，我们需要明确APP的类型呢？因为不同的APP类型，其日活定义会存在差异。

2. 确认日活定义

那么，不同类型的APP，其日活定义会有什么区别呢？

APP类型 日活定义 参考产品
工具 每日登陆？ 墨迹天气
社交 用户在平台上与他人产生至少一条互动（发动态、点赞、评论、转发、私聊）？ soul
电商 用户完成一笔订单？ 拼多多
内容 用户看多少分钟的内容？ 微信读书

3. 确认日活下降5%的定义？

在我们开始确认数据的时候，我们需要先明白，产品目前的所处阶段：

  1. 探索
  2. 成长
  3. 成熟
  4. 衰退

3.1 确认数据本身

数据本身，需要我们思考什么呢？

  • 数据的真实性？

    1. 数据的来源/统计口径？

  • 数据的绝对值

    1. 5%是如何计算出来的？

    2. 基数是多少？

3.2 数值同比

同比是考虑月份的影响，比如判断去年1月和今年1月的数据。

  • Y
    1. 产品走下坡路
    2. 市场整体行情下滑
    3. 市场竞争格局变化
  • N

3.3 数值环比

环比是周/月环比。近一周/月和上一周/月的日均DAU数值差异。

  • Y
    • 内部
      1. 产品：产品是否最近有大的更新
      2. 运营：运营是否有大的运营动作
      3. 品牌：最近是否有负面新闻
      4. 市场：在市面上的投放最近是否有变化
      5. 技术：是否出了什么Bug
    • 外部
      1. 竞争对手是否有大的动作
  • N

3.4 数值横向比较

横向比较，是指和市面上其他产品的比较。如无法获得相关数据，则不从这条分析。

4. 分析影响日活的因素

4.1 工具类：

  • 缓慢下滑
    1. 产品走下坡路
      1. 采访留存用户
      2. 采访流失用户
    2. 市场整体行情不行
      1. 如2018年互联网经济危机
    3. 市场竞争格局变化
      1. 如京东迎来了拼多多
  • 断崖式下滑
    1. 内部
      1. 产品
      2. 运营
      3. 品牌
      4. 市场
      5. 技术
    2. 外部
      1. 竞争对手是否有大的工作

备注：社交、电商、内容的分析思路同工具类
